웹브라우저

    - 웹서버에 원하는 자원을 요청하는 클라이언트 프로그램이다.
    - HTML 내용을 읽어서 눈에 보이는 웹페이지를 만드는 역활이다.
    - 웹 애플리케이션에 대한 사용자 인터페이스 - UI(User Interface)를 제공
    - 웹 애플리케이션 - 브라우저에서 이용할 수 있는 응용 프로그램이다.

HTML(Hyper Text Markup Language) 

    - 웹 브라우저 상에 정보를 표시하기 위한 마크업 심볼들의 집합
    - 마크업
      : 태그등을 이용하여 문서나 데이터의 구조를 명시하는 언어
    - 태그
      : 태그(tag)는 어떤 정보를 메타데이터로 부여된 키워드로 분류
    - 메타데이타
      : 데이터를 설명하는 데이터임     

HTML 버전
    - HTML5(2012) <- XHTML1.0(2000) <- HTML 4.01(1999) .... HTML(1991)

HTML 특징
    - 태그는 대소문자 구분 없음, <DIV></DIV>, <div></div>
    - 확장자는 html 또는 htm 을 사용
    - 엔터, 스페이스바, 탭을 무시한다.
      <span>H 
      
      
            i</span> -> H i

HTML 주요 용어
    - 요소(Element)   
      : 시작태그 + 콘텐츠(내용) + 종료태그
      : 시작태그 부터 종료태그 까지의 모든 명령어를 요소라고 함
      : <title>연습</title>
      : title : 엘리먼트 이름 
      : 연습 : 엘리먼트 내용

      * 주의 : 종료태그를 사용하지 않은 경우 브라우저마다 다른 처리를 할 수 있음        

    - 태그  (Tag)
      : "<"와 ">" 로 묶인 명령어
      : 형태 <태그명>....</태그명>
      : 종료태그는 태그명 앞에 "슬래쉬(/)"를 붙임
      : 시작과 종료를 같이 사용하는 경우(<img />)

    - 속성(Attribute)
      : 속성은 시작태그에만 적용할 수 있다.
      : 각각의 시작태그에 필요에 따라서 적용 가능
      : 속성을 시작태그에 여러개 설정할 수 있다.
      : <div id="aaa" class="bbb"></div>  
      : id, class : 속성이름
      : aaa, bbb : 속성 값
